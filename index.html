So i Source

＃So i Sourceとは

プログラミングを基礎から学習できる、オンラインカリキュラムです！

  • 未経験エンジニアのキャリア育成

    IT業界未経験者をWEBエンジニア、デザイナーとして自社で育成するカリキュラム／教育体制が整っており、0からのスタートを応援します。

  • キャリアコンサルティング

    「やりたい」事を実現できるレベルにまで引き上げていきます。
    「So i Source選んでよかった」を目指します。
    すでにエンジニア経験のある方はもちろん、未経験の皆さんも安心してキャリアアップできる環境に注力し、皆さんの成長、キャリアアップを全力でサポートします。

  • 「そう、私がソース」

    自分でソースを書けるようになってほしい、私のソースだ、私がソースだ！と言えるようになってほしいという感じの意味。
    ＋醤油みたいに万能なプログラミングカリキュラムです。

＃CURRICULUMについて

4つのコースを用意しております。ご自身にあったコースをお選びください！

  • HTML, CSS, Javascript

    フロントエンドエンジニアとして必要となる、web制作の基礎を学習できます。
    カリキュラム内容：HTML, CSS, Javascript, jQuery, SASS, ejsなど
    基礎〜応用までしっかり学習したのち、ポートフォリオ制作を行います。

    Webアプリケーションの開発に興味がある方にはマストのコースと言っても過言ではないでしょう。

  • Web Design

    Webデザイナーとして必要となる、デザインの知識と対応力を身につけます。 カリキュラム内容：Web Designにおける、ツールの使用方法〜実際の制作まで、一連の流れを学習・体感することができます。
    カリキュラム中には多数課題があり、より実践的な学習が可能。
    ポートフォリオ制作では実際に0からデザインを行います。
